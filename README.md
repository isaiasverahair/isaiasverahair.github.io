# 💇 Isaías & Vera - Cabeleireiros Premium

## 🌟 Sobre

Website profissional para **Isaías & Vera Cabeleireiros**, um salão de beleza com **42 anos de excelência** em tratamentos capilares especializados, localizado em Nova Friburgo, Rio de Janeiro.

O site apresenta um design moderno e elegante, focado em destaque de serviços premium, avaliações de clientes, galeria de trabalhos e integração com WhatsApp para agendamentos.

---

## ✨ Características

✅ **Design Responsivo** - Adapta-se perfeitamente em todos os dispositivos  
✅ **Interface Premium** - Paleta de cores sofisticada (nude, rose-gold e ouro)  
✅ **Integração WhatsApp** - Agendamento direto via WhatsApp  
✅ **Animações Suaves** - Transições visuais elegantes e fluidas

## 🔧 Personalizações

Para modificar o site, você pode:

1. **Alterar cores** - Edite as variáveis CSS em `styles.css`
2. **Atualizar serviços** - Modifique a seção de serviços em `index.html`
3. **Adicionar galeria** - Insira novas imagens na pasta `/img`
4. **Atualizar WhatsApp** - Altere o número no atributo `href` dos links de WhatsApp
